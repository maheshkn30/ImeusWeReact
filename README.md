# ImeusWeReact

A full-stack web application built with React and Express.js that manages member information. The project combines a React frontend with a Node.js/Express backend and MongoDB database integration.

## Key Features

- **Member Management**: Add and manage team members through a dedicated interface
- **Full-Stack Architecture**: React frontend with Express.js backend
- **Database Integration**: MongoDB support via Mongoose for persistent data storage
- **API Communication**: Axios for seamless client-server communication
- **Responsive Design**: CSS-styled components with routing capabilities
- **Development Tools**: Concurrent development server with hot-reload support

## Tech Stack

- **Frontend**: React 18.3.1, React Router DOM 6.23.1
- **Backend**: Express.js, Node.js
- **Database**: MongoDB with Mongoose ODM
- **HTTP Client**: Axios
- **Development**: Concurrently for running both servers simultaneously
- **Build Tool**: React Scripts

## Main Components

- **Header**: Navigation component
- **Addmember**: Page to add new members to the system
- **Server**: Node.js/Express backend
- **Models**: AddmembersModel for data structure

## Available Scripts

- `npm run dev` - Run both React and server concurrently
- `npm run start:react` - Start React development server
- `npm run start:server` - Start Express backend server
- `npm run build` - Build optimized production bundle

## Getting Started

1. Install dependencies: `npm install`
2. Run the application: `npm run dev`
